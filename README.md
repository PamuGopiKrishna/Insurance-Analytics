# Insurance Analytics Project

## 📊 Overview
This project analyzes insurance claims and policyholder data to identify risk patterns and business insights.

## 🛠 Tools Used
- SQL
- Excel
- Power BI
- Tableau

## 📂 Project Structure
- dataset → raw data
- excel → dashboards
- powerbi → reports
- sql → queries
- tableau → dashboards

## 📈 Key Insights
- High claim frequency in certain regions
- Age group impact on claims
- Policy type vs claim amount
